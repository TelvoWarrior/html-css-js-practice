All CSS Grid Properties

  • column-gap - Specifies the gap between the columns
    Свойство определяет размер зазора только между колонками.
  • gap - A shorthand property for the row-gap and the column-gap properties

    Эта краткая запись позволяет определить размер зазора как между колонками, так и между строками.

    Возможны два варианта записи.

    Первый заключается в том, что бы указать один размер и этот размер будет использоваться для определения зазора и между колонками, и между строками.

    Второй вариант заключается в том, что бы указать два размера. При этом первый размер будет определять размер зазора между строками, а второй - между колонками.

    Значения отделены друг от друга пробелом, а не запятой.
  • grid - A shorthand property for the grid-template-rows, grid-template-columns, grid-template-areas, grid-auto-rows, grid-auto-columns, and the grid-auto-flow properties
    Я затрудняюсь описать функционал данного свойства.
  • grid-area - Either specifies a name for the grid item, or this property is a shorthand property for the grid-row-start, grid-column-start, grid-row-end, and grid-column-end properties
    Свойство может использоваться в двух разных способах.
    Первый - оно может определять название грид айтема. Речь идет о каком то разделе на странице. Например, хэдер, футер, главная часть, боковая панель и т.д.
    Второй вариант использование этого свойства - это краткая запись для свойств: the grid-row-start, grid-column-start, grid-row-end, and grid-column-end properties.
    Эти свойства записываются через слэш в порядке, указаном выше.
  • grid-auto-columns - Specifies a default column size
    Не уверен, что полностью понимаю это. Данное свойство устанавливает ширину по-умолчанию для колонок.
  • grid-auto-flow - Specifies how auto-placed items are inserted in the grid
    Свойство определяет в каком порядке будут добавляться новые айтемы в контейнер. Свойство может принимать следующие значения: row, column, dense, row dense, column dense
    row - новые айтемы буду добавляться слева направо
    column - новые айтемы будут добавляться сверху вниз
    dense - я затрудняюсь описать действие данного значения.
    row dense - я затрудняюсь описать действие данного значения.
    column dense - я затрудняюсь описать действие данного значения.
  • grid-auto-rows - Specifies a default row size
    Не уверен, что полностью понимаю это. Cвойство устанавливает высоту по-умолчанию для строк.
  • grid-column - A shorthand property for the grid-column-start and the grid-column-end properties
    Cвойство является краткой формой записи определяющей с какой колонки начинается и на какой колонке заканчивается определяемый грит айтем.
  • grid-column-end - Specifies where to end the grid item
    Cвойство определяет колонку на которой должен закончится определяемый грид айтем.
  • grid-column-gap - Specifies the size of the gap between columns
    Насколько я понял, это старое свойство. В настоящий момент его заменяет свойство column-gap. Cвойство определяет размер зазора между колонками.
  • grid-column-start - Specifies where to start the grid item
    Cвойство определяет колонку с которой должен начаться определяемый грид айтем.
  • grid-gap - A shorthand property for the grid-row-gap and grid-column-gap properties
    Cтарое свойство. В настоящий момент его заменяет свойство gap. Cвойство позволяет определить размер зазора как между колонками, так и между строками.
  • grid-row - A shorthand property for the grid-row-start and the grid-row-end properties
    Cвойство является краткой формой записи. Оно определяет с какой строки должен начаться и на какой строке должен закончится определяемый грид айтем.
  • grid-row-end - Specifies where to end the grid item
    Cвойство определяет на какой строке должен закончится определяемый грид айтем.
  • grid-row-gap - Specifies the size of the gap between rows
    Cтарое свойство. В настоящий момент его заменяет свойство row-gap. Cвойство определяет размер зазора между строками.
  • grid-row-start - Specifies where to start the grid item
    Cвойство определяет с какой строки должен начаться определяемый грид айтем.
  • grid-template - A shorthand property for the grid-template-rows, grid-template-columns and grid-areas properties
    Неполное описание. Cвойство является краткой записью сразу двух других свойств: grid-template-rows, grid-template-columns. Сначала через пробел указываются все значения строк, а затем ставится слэш и так же через пробел указываются все значения колонок. grid-areas properties - не разбирался
  • grid-template-areas - Specifies how to display columns and rows, using named grid items
    Я затрудняюсь описать функционал данного свойства.
  • grid-template-columns - Specifies the size of the columns, and how many columns in a grid layout
    Cвойство указывает количество и ширину колонок в grid layout. Значения отделены друг от друга пробелами, каждое значение указывает ширину соответствующей колонки.
  • grid-template-rows - Specifies the size of the rows in a grid layout
    Cвойство указывает количество и высоту строк в grid layout. Значения отделены друг от друга пробелами, каждое значение указывает высоту соответствующей строки.
  • row-gap - Specifies the gap between the grid rows
    Cвойство определяет размер зазора между строками.
